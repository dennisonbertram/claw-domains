# Claw Domains - Documentation Index

Central documentation hub for the Claw Domains project.

## Folders

| Folder | Description |
|--------|-------------|
| [research/](research/) | Research findings on technologies, protocols, and approaches |
| [design/](design/) | Architecture diagrams, design documents, and system design |
| [explorations/](explorations/) | Exploratory notes, spikes, and proof-of-concept findings |
| [plans/](plans/) | Feature implementation plans with step-by-step checklists |
| [logs/](logs/) | Engineering log, observation log, and systems documentation |
| [runbooks/](runbooks/) | Step-by-step operational guides for testing, deployment, etc. |
| [implementation/](implementation/) | Implementation notes and code change documentation |
| [investigations/](investigations/) | Deep-dive investigations and debugging sessions |
| [decisions/](decisions/) | Technical decision records (TDRs) with rationale |
